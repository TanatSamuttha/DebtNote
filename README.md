# DebtNote

This web application was build to fix a pain point that you can't remember how much money are your friends borrowed. And when you and your friends check an amount in private record, the amount is not same.
So, this web application will solve this problem by sharing an amount of money to the creditor and debtor to have the same record.

## TechStack

Backend language: Go
API Framework: Fiber
ORM: GORM
Database: Postgres
